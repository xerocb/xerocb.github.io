HTML & CSS Cheat Sheet

HTML Table Tags

Tag Name Description
<table> Table Defines an HTML table. All content for the table is stored within the table element.
<thead> Table Head Defines the head of a table. This can be used in combination with CSS styles to visually distinguish the headings of a table from its data.
<tbody> Table Body Defines the body of a table. This is where table data lives.
<tr> Table Row Defines a row for a table. Rows typically contain multiple columns, created using the <th> and <td> tags.
<th> Table Header Defines a column for a header row. By default the text will be bold to distinguish headers from table data.
<td> Table Data Defines a column for table data.

HTML Semantic Tags

Semantic tags define different parts of a webpage, clearly outlining their purpose to both the browser and the developer.

Tag Name Description
<section> Section Defines a section, or a "thematic grouping of content, typically with a heading."
<article> Article Defines an article, which should contain independent, self-contained content.
<header> Header Defines a header, typically containing introductory content.
<nav> Navigation Defines a section specifically used to contain a set of navigation links.
<footer> Footer Defines a footer, usually displayed at the bottom of the page and often containing contact information.
<aside> Aside Defines an aside section, often placed in a sidebar, which should be indirectly related to its surrounding content.
<figure> Figure Defines a figure, typically used for images, diagrams, and the like.
<figcaption> Figure Caption Defines a caption for a figure. Must be defined within a <figure> element.

CSS Selectors

Type Example Description Specificity
element p Selects all <p> elements. (0, 0, 1)
.class .intro Selects all elements with class="intro". (0, 1, 0)
#id #firstname Selects the element with id="firstname". (1, 0, 0)
.class1.class2 .name1.name2 Selects all elements which have both name1 and name2 in their class attribute (separated with whitespace). (0, 2, 0)
.class1 .class2 .name1 .name2 Selects all elements with class="name2" inside elements with class="name1". (0, 2, 0)
element.class p.intro Selects all <p> elements with class="intro". (0, 1, 1)
element, element p, h1 Selects all <p> and <h1> elements. (0, 0, 1)
